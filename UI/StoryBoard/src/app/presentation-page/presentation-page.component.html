Sobre nós

Este projeto de Trabalho de Conclusão de Curso (TCC) tem como objetivo desenvolver um sistema de gerenciamento de tarefas baseado no método Kanban. O Kanban é um sistema visual que auxilia no planejamento, acompanhamento e controle das atividades em um fluxo de trabalho.
