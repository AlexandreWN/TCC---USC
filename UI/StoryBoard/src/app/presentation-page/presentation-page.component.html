Sobre nós

Nosso Trabalho de Conclusão de Curso (TCC) tem como foco o desenvolvimento de um sistema de gerenciamento de tarefas fundamentado no método Kanban. O objetivo principal é criar uma plataforma que emprega o método Kanban, um sistema visual altamente eficiente, para aprimorar o planejamento, acompanhamento e controle das atividades em fluxos de trabalho. O Kanban, originado no Japão, é uma metodologia reconhecida por sua abordagem visual e pragmática para organizar tarefas. Por meio de colunas e cartões, permite a fácil identificação do progresso das atividades, promovendo uma visão clara do fluxo de trabalho. Este projeto busca explorar e implementar os princípios do Kanban para oferecer uma solução flexível e eficaz. Pretendemos criar uma ferramenta que possibilite aos usuários uma gestão simplificada e transparente de suas tarefas, aumentando a produtividade e melhorando a eficiência dos processos de trabalho.
